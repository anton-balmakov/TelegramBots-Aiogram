# Telegram Bots (Aiogram)

Коллекция Telegram-ботов на Python (aiogram): ответы на команды, работа с медиа, ссылки, интеграции и API-запросы.

## Что есть
- несколько ботов под разные задачи (учебные и практические)
- примеры работы с API / парсингом
- обработка сообщений, фото/видео, кнопки (если есть)

## Стек
Python, aiogram, requests (и др. по проекту)

## Запуск
В каждом боте свой `BOT_TOKEN` (через `.env` или переменные окружения).
